üîß Config File Structure

Your server is configured using a .conf file. Here's a sample:


server {
  listen 8080;
  server_name localhost;

  location / {
    root ./www;
    index index.html;
    autoindex on;
    allow_methods GET POST;
  }

  error_page 404 /404.html;
  client_max_body_size 10M;
}

‚úÖ Supported Directives

  • listen: Port the server listens on
  • server_name: Optional hostname
  • root: Folder for this location
  • index: Default file to serve
  • autoindex: Enables directory listing
  • allow_methods: Permitted HTTP verbs
  • error_page: Custom error response
